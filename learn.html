1.What is a Raspberry Pi?

Raspberry Pi is a low cost, credit card sized, computer. It is a fully featured, internet connected Linux computer that is easy to embed into prototypes.

Raspberry Pi works as a platform for interactive devices and the internet of things. GPIO pins and HAT extensions make it easy to add sensors, physical controls, outputs and interfaces to your projects.

Should I Get A Raspberry Pi or Arduino?

Raspberry Pi is great for handling video or graphic interfaces. It can deals with projects that process lots of data. It works for computer vision and other performance-dependant processes.

Aruino is great for controlling lots of sensors, small LCDs, motors, and Analog sensors. It is ideal for low power applications(ie. battery powered projects).

2. Getting Started

Let's get your Raspberry Pi set up


________________________________________________________________________________________________

What You Will Need

Hardware

A Raspberry Pi

A Desktop Computer or Laptop

A USB Keyboard & Mouse

Micro USB Cable (Power Supply)

A HDMI Cable

A Micro SD Memory Card or SD Memory Card

Software

Raspbian installed via NOOBS


________________________________________________________________________________________________

Remember To Label Your Pi

Everyone’s Raspberry Pi and power supply looks the same. Labeling will help you identify which one is yours from your peers.


________________________________________________________________________________________________

Something To Keep In Mind

Make sure that your circuits are complete before powering. With Raspberry Pi, this also ensures all peripherals initialize properly while it boots up. Make sure your Pi is off while making changes to attached circuits.


________________________________________________________________________________________________

Set Up For The First Time

Step 1. Install the Micro SD memory card in the back.

Step 2. Plug in the HDMI cable.

Step 3. Plug in a USB Mouse.

Step 4. Plug in a USB Keyboard.

Step 5. Plug in Micro USB cable.

Step 6. Install Raspbian from the screen (NOOBS). NOOBS allows us to choose an operating system to install. These cards come preinstalled, but a fresh setup would require moving some files onto the SD card. Choose Raspbian and click Install.

Step 7. Wait for installation to complete

Step 8. The setup wizard will walk you though setting location (for time), keyboard type (usually US layout), and changing the system password.

Congratulation You've Finished Set Up. The setup wizard settings can be changed later, through the “Raspberry Pi Configuration” menu item
